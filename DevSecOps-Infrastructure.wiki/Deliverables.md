# Required Deliverables (Screenshots)

The assignment expects several screenshots to prove that the infrastructure and toolchain are running correctly.

All screenshots should be stored in the main repository under:

```text
Execution Example Screenshot/
```

## Required Screenshots

- Successful `terraform apply`
- AWS VPC view
- Public and private subnets
- EC2 instances list
- Jenkins web UI
- Nexus web UI
- SonarQube web UI
- `kubectl get nodes` showing Ready worker nodes

These screenshots should match the resources created by Terraform and configured by Ansible.
